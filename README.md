# Email-Of-Members-AD-Groups

Script for search email addresses of users ad groups for convenient copying into Outlook.

Example:
Введите ввод: gr-zabbix-test

Output:
Ex.ample@domain.local Am.ple@domain.local Pl.aybook@domain.local

Setup:
C:/Program Files/WindowsPowerShell/Modules/Search-EmailADGroup/Search-EmailADGroup.psm1
